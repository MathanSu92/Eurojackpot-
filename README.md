# Eurojackpot Tippgenerator

Eine kleine Desktop-App in Python, die zufällige, gültige Tipps im Format **5 aus 50 + 2 Eurozahlen aus 12** erzeugt.

> Hinweis: Lotterieziehungen sind zufällig. Diese App kann keine Gewinnzahlen vorhersagen; jeder gültige Tipp hat dieselbe Gewinnchance.

## Voraussetzungen

- Python 3.9 oder neuer
- `tkinter` (bei vielen Python-Installationen bereits enthalten)

## Lokal starten

```bash
git clone https://github.com/DEIN-GITHUB-NAME/eurojackpot-tipgenerator.git
cd eurojackpot-tipgenerator
python app.py
```

Unter Linux kann `tkinter` bei Bedarf installiert werden:

```bash
sudo apt install python3-tk
```
